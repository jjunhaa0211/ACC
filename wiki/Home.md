# ACC CI/CD Automation Wiki

`acc-ci`는 프로젝트 유형과 무관하게(우선 Node/Python/Go 지원) 다음을 자동화하기 위한 CLI+라이브러리입니다.

- 테스트 코드 스캐폴딩
- mock 데이터 스캐폴딩
- CI 워크플로 자동 생성
- 로컬/CI 동일 검증 파이프라인 실행

## 핵심 가치

1. 한 명령으로 시작 가능
2. 팀 표준에 맞춰 설정 파일 커스터마이징 가능
3. mock 계약 검증을 통해 회귀를 빠르게 탐지
4. CI와 로컬 결과의 차이를 최소화
